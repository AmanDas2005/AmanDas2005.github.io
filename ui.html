UI/UX Design

Figma, Adobe Photoshop, Adobe Illustrator, Microsoft PowerPoint, Django

Building on the skills I developed in Visual Design, I designed and implemented user interfaces for two websites.

The first was a Georgia Tech–themed movie store, where I used the school's color palette to create a cohesive and recognizable design.

The second, TravelMate, focused on trip planning, with an interface inspired by notes, paper, and labels to create an intuitive and organized user experience.

Across both projects, I applied core design principles to produce interfaces that are both visually engaging and user-centered.
