JumpRace Micro

Visual Studio Code (Coding Language: C), Bandlab, mGBA, Audacity, Tiled, Usenti, Piskel

For my Media Device Architecture course, I designed an original game for the GameBoy Advance titled JumpRace Micro.

Inspired by the fast-paced structure of WarioWare, the game features a series of short, varied minigames, each presenting a unique challenge.

The levels incorporate a mix of platforming and shooting mechanics, with several drawing inspiration from classic arcade-style gameplay.

This approach allowed me to create a dynamic and engaging experience while exploring different gameplay styles within a single project.
